Belvedere Glacier — Long-Term Monitoring

STAC Catalog  ·  14 photogrammetric surveys  ·  1977–2024

This catalog indexes the open dataset of the Belvedere Glacier (Monte Rosa, Italian Alps), including UAV and historical aerial photogrammetric surveys. Each item provides a Cloud-Optimized GeoTIFF DSM, a COG orthophoto, and a COPC point cloud, all streamed directly from Zenodo.

Catalog entry point

https://franioli.github.io/belvedere-open-data/catalog.json

Quick start (Python)

import pystac, rasterio

catalog = pystac.Catalog.from_file(
    "https://franioli.github.io/belvedere-open-data/catalog.json"
)
collection = catalog.get_child("belvedere-monitoring")

# List all surveys
for item in sorted(collection.get_items(), key=lambda i: i.datetime):
    print(item.id, item.datetime.year, item.properties.get("platform"))

# Stream a DSM overview (no full download)
item = collection.get_item("belv_2022_uav")
with rasterio.open(item.assets["dsm"].href) as src:
    overview = src.read(1, out_shape=(src.height // 8, src.width // 8))
